# emergent-ai

A React-based portfolio site built with Create React App (CRACO), Tailwind CSS, Radix UI, and Framer Motion.

## Tech stack

- **React 19** with Create React App (CRACO)
- **Tailwind CSS** for styling
- **Radix UI** for accessible components
- **Framer Motion** for animations
- **Lucide React** for icons

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Yarn](https://yarnpkg.com/) (v1.x; project uses `packageManager` in `package.json`)

## Setup

Install dependencies:

```bash
yarn install
```

## Development

Run the app in development mode with hot reload:

```bash
yarn start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Build

Create a production build:

```bash
yarn build
```

Output is written to the `build/` directory and is suitable for static hosting (e.g. GitHub Pages).
